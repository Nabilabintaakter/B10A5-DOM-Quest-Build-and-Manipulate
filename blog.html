Important FAQ

What is the Document Object Model (DOM)?

Document Object Model (DOM) is an application programming interface for HTML document. JavaScript actually interacts with the html webpage through this DOM. And can manipulate the html elements easily.

How do you select an element from the DOM?

There are some important methods in JavaScript by which we can select a particular html element from the DOM. These are:

  • 1. document.getElementById() : For selecting an element by id.
  • 2. document.getElementsByClassNames() : For selecting elements by class name.
  • 3. document.getElementsByTagName() : For selecting elements by tag name.

We can also select elements by css selectors from the DOM. These are:

  • 4. document.querySelector(): for selecting the first element which matches the provided css selector.
  • 5. document.querySelectorAll(): for selecting all the elements matching the provided css selector.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a method by which we add an event listener to the parent node instead of its child nodes for preventing event bubbling. By doing this we can escape adding many event handlers to the child nodes.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attribute, setAttribute() method is used. It has 2 parameters: attribute name and value. And we can use the style property for styling an element dynamically from DOM.
